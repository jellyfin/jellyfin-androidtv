Vaše média, za vašich podmínek.

Projekt Jellyfin je bezplatný multimediální server s otevřeným zdrojovým kódem. Bez poplatků, sledování či nekalých zájmů. Náš bezplatný server umožňuje spravovat zvuk, video, fotky a další média na jednom místě.

Abyste mohli aplikaci používat, musíte mít nastavený server Jellyfin. Další informace naleznete na webu <a href="https://jellyfin.org/" target="_blank">jellyfin.org</a>.

Se serverem Jellyfin můžete:

* Sledovat živé i nahrané televizní vysílání (vyžaduje dodatečný hardware/služby)
* Streamovat do zařízení Chromecast ve vaší síti
* Streamovat vaše vlastní média do vašeho zařízení se systémem Android
* Zobrazit svou sbírku médií pomocí snadno použitelného rozhraní

Toto je oficiální aplikace Jellyfin pro systém Android TV. Děkujeme, že používáte Jellyfin!
